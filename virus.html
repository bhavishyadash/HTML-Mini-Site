What is a Virus?

A computer virus is malicious code that attaches to a host program or file and replicates when that host is executed. Variants include file infectors, macro viruses, and boot sector attacks.

Propagation & Payloads

Spread often relies on user action (opening attachments, running cracked software), removable media, or exploits. Payloads range from nuisance messages to data theft and ransomware. Defense layers include least privilege, updates, backups, and behavior-based detection.
