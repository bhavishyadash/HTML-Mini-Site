Paradigms

Languages encourage ways of thinking. Procedural (C) favors step-by-step routines. Object-oriented (Java, C#) organizes data and behavior together. Functional (Haskell, Elixir) leans on pure functions and immutability. Many modern languages are multi-paradigm (Python, JavaScript).

Compiled vs. Interpreted

  • Compiled: Translate to machine code ahead of time (C, Rust).
  • Interpreted: Run via a VM/engine at runtime (Python, JS).
  • Hybrid/JIT: Mix approaches for speed and portability (Java, .NET).

Type Systems

Static types catch shape errors early; dynamic types trade some safety for speed of iteration. Modern compilers add type inference to reduce boilerplate (e.g., auto in C++, var in C#, and TypeScript atop JS).
